---

## Target state

Each repository should have:

- `.agentplane/WORKFLOW.md` validated by schema + strict template checks.
- `.agentplane/workflows/last-known-good.md` snapshot.
- `agentplane doctor` passing without `ERROR`.

## Repair steps

1. Apply safe doctor fixes:

```bash
agentplane doctor --fix
agentplane doctor
```

2. If doctor reports that the active workflow contract needs maintainer-level recovery, inspect the
   hidden recovery surface:

```bash
agentplane help --all
```

3. Re-run `agentplane doctor` after the recovery command reports success.

## Kill-switch (emergency only)

Set environment variable to disable workflow contract enforcement temporarily:

```bash
export AGENTPLANE_WORKFLOW_ENFORCEMENT=off
```

Effects:

- `agentplane preflight` skips workflow validation.
- `agentplane doctor` skips workflow contract checks.

Recovery:

1. Remove the variable.
2. Run `agentplane doctor`.
