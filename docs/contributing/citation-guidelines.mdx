# Citation guidelines

## Rules

- Link to primary sources when possible.
- Use vendor docs, standards docs, research papers, or respected engineering publications before
  random blogs.
- Do not overquote. Summarize and link.
- Do not imply endorsement by referenced authors.
- Use references to clarify concepts, not to pad authority.
- Prefer 3-5 strong references per concept page over 20 weak links.

## How to add a source

Add the source to `website/src/data/referenceSources.ts`, then render it with
`FurtherReading`. This keeps citations consistent and prevents duplicated link lists.
