# Release Notes - v0.6.4

## Summary

- AgentPlane v0.6.4 is a patch release focused on branch_pr lifecycle correctness, release readiness gates, and context workflow reliability after the v0.6.3 line.

- No breaking migration is required for existing v0.6 users.
