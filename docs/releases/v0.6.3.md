# Release Notes - v0.6.3

## Summary

- Agentplane v0.6.3 is a patch release for the v0.6 line after the local task registry cleanup, context assimilation work, release hardening, and website/docs refresh landed on protected main.
- The release keeps the public package line on v0.6 while tightening branch_pr closure, evaluator quality gates, release candidate preparation, and publication evidence.
- Context workflows gain stronger maximum-assimilation and wiki-scaffold behavior, while docs and website IA are refreshed for agent-first onboarding.
- No breaking migration is required for existing v0.6 users.
