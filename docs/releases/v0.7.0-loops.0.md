# Release Notes - v0.7.0-loops.0

## Summary

- This branch-local prerelease introduces AgentPlane loops as a first-class
  controlled feedback-cycle surface.
- It adds built-in loop templates, project-local loop validation, loop route
  planning, loop explanation, and dry-run LoopRun evidence.
- It deliberately does not enable non-dry-run external agent execution through
  `agentplane loop run`; that remains a future trust-boundary task.

## Added

- Added `agentplane loop list`, `loop show`, `loop explain`, `loop plan`,
  `loop validate`, `loop run --dry-run`, and `loop step --dry-run`.
- Added LoopSpec, LoopRun, event, and decision TypeScript contracts.
- Added built-in loops for TDD repair, CI repair, docs sync, security review,
  context harvest, workflow recovery, evaluator quality gates, and loop
  improvement proposals.
- Added dry-run LoopRun artifacts under
  `.agentplane/tasks/<task-id>/runs/<run-id>/`.
- Added `loop-spec.schema.json` and a public spec example.
- Added developer and reference documentation for loop semantics, governance,
  selection, and evidence layout.

## Governance

- Loop execution is separate from loop modification.
- Loop modification is separate from trust activation.
- Project-local loop specs are validated before use.
- Non-dry-run loop execution fails closed in this version.
