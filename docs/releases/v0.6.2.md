# Release Notes — v0.6.2

## Summary

- AgentPlane v0.6.2 is a patch release for the v0.6 line that includes route-decision CLI guidance, release publication hardening, CodeQL guardrail fixes, context wiki refinements, README and website polish, and dependency maintenance.
- The release narrows branch_pr operational ambiguity by documenting and testing route-decision surfaces while preserving the existing protected-main PR workflow.
- Release and publication evidence is stricter: platform publication checks, release-ready manifests, README header generation, and external-channel proof are easier to audit.
- No breaking migration is required for existing v0.6 projects; blocked major dependency upgrades remain outside this release.
