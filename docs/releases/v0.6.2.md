# Release Notes - v0.6.2

## Summary

- AgentPlane v0.6.2 is a patch release for the v0.6 line, built from the current protected main after the DX, release-hardening, security, and workflow fixes landed.
- The release improves branch_pr operations with route-decision commands, guided task shortcuts, a first-success demo command, and stronger feedback for local quality checks.
- Publication evidence is stricter across release notes, platform publication checks, release-ready manifests, README headers, and post-merge proof surfaces.
- No breaking migration is required for existing v0.6 users; failed semver-major dependency upgrades remain follow-up work outside this patch release.

## Upgrade Notes

- Existing AgentPlane v0.6 projects can upgrade to v0.6.2 as a patch release without changing workflow configuration.
- The new guided commands are additive shortcuts over the existing lifecycle primitives, so existing automation can continue using the lower-level commands.
- The branch_pr route-decision commands are operational guidance surfaces; they do not replace the protected-main PR merge route.
- Semver-major dependency updates that still fail checks, including zod 4, execa 9, eslint-plugin-n 18, ESLint 10, and TypeScript 6 lanes, are intentionally not required for this release.
- Projects with generated local artifacts should rerun their normal AgentPlane verification and bootstrap commands after upgrading.

## Verification
