# Release Notes - v0.6.2

## Summary

- AgentPlane v0.6.2 is a patch release for the v0.6 line, built from the current protected main after the DX, release-hardening, security, and workflow fixes landed.
- The release improves branch_pr operations with route-decision commands, guided task shortcuts, a first-success demo command, and stronger feedback for local quality checks.
- Publication evidence is stricter across release notes, platform publication checks, release-ready manifests, README headers, and post-merge proof surfaces.
- No breaking migration is required for existing v0.6 users; failed semver-major dependency upgrades remain follow-up work outside this patch release.
