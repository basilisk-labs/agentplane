# Release Notes — v0.6.1

## Summary

- Agentplane v0.6.1 consolidates the post-v0.6.0 hardening wave into a patch release focused on release evidence, cloud backend safety, task lifecycle correctness, and public documentation polish.
- The release improves branch_pr close-tail reliability, release-note validation, publication proof, and diagnostics around feedback issues, remote checks, task readback, and cloud sync behavior.
- Website, README, blog, navigation, and context-announcement updates are included so the public surface matches the v0.6 context-management positioning.
- All changes remain patch-scoped: no breaking CLI migration is required for existing Agentplane projects.
