# Release Notes — v0.2.26

- Improved `init` recovery behavior: when `.agentplane` config/backend conflicts already exist, missing `AGENTS.md` and agent templates are now restored before conflict reporting.
- Added regression coverage for the `init` conflict scenario where policy files are missing in an existing repository.
- Hardened task scanning and bootstrap performance paths with faster stale-build checks and stricter task-read diagnostics.
- Expanded workflow/core command reliability with better reconcile guards, reason-code taxonomy, and clearer decode output in CLI and JSON errors.
- Continued release and workflow hardening across hooks, docs synchronization, and generated CLI references.
