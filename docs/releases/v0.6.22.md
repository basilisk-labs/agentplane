# Release Notes - v0.6.22

## Summary

- This patch release completes the context-extraction quality program on protected `main`, including compact schema-complete task packs, byte-bounded batching, duplicate suppression, transactional writes, and stricter cross-surface integrity.
- It improves generated wiki granularity and connectivity through richer entity references, source spans, graph validation, topology reports, and maximum-assimilation migration support for both agents and people.
- It reduces maintainability debt by splitting context, routing, task-command, runtime, backend, and oversized test hotspots while lowering the verified Knip baseline.
- It hardens branch_pr lifecycle, integration-queue recovery, hosted closure, release routing, website lint, and evaluator evidence so the package cut can be reproduced from the exact merged commit.
