# Website success metrics

## Primary metrics

- Quickstart page visits.
- Install command copies.
- GitHub star clicks.
- Docs-to-GitHub clicks.
- Example page visits.

## Secondary metrics

- Organic search impressions.
- Branded search growth.
- AI-referred sessions.
- Concept page traffic.
- Returning docs visitors.

## Events

```text
quickstart_cta_click
copy_install_command
copy_run_command
navbar_github_star_click
view_example
view_harness_guide
view_context_guide
view_traces_guide
```

Do not optimize only for pageviews. The site should drive CLI usage and GitHub engagement while
keeping analytics lightweight and privacy-respecting.
