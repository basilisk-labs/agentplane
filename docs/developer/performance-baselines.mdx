## CLI Cold Path

Baseline captured on 2026-04-20 from the repo-local CLI:

```bash
bun run bench:cli:cold --runs 5 --warmups 1
```

Environment and dataset:

- Checkout: `/Users/densmirnov/Github/agentplane`.
- CLI: `packages/agentplane/bin/agentplane.js`.
- Mode: `cli_cold_path_v1`.
- Runs: 5 timed runs per command after 1 warmup.
- Task dataset: full framework task store in this checkout, including 1900+ historical tasks.
- Working tree note: the benchmark saw a non-clean tracked-only preflight state as `false` in the
  command output preview context; this baseline should be used for relative local comparisons, not
  cross-machine claims.

Use this baseline after packaging, subpath export, startup, or task-index changes. Compare the same
