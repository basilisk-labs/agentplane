# AgentPlane documentation

AgentPlane turns AI-agent work into reviewable Git evidence: task intent, plan, verification, local
context, Agent Change Record, and closure.

Humans install and initialize AgentPlane. Coding agents operate the lifecycle. Reviewers inspect the
evidence in Git.
