# Agentplane documentation

Agentplane is an agent-agnostic harness for governed coding-agent work. The agent writes code,
uses the CLI, and records evidence; the human sets tasks, guardrails, and review boundaries through
the harness.

## Documentation layers

- **Start**: install, initialize, and inspect the first artifacts.
- **Context**: maintain project knowledge that humans can review and agents can reuse.
- **Agent workflows**: hand approved work to any coding agent and review the evidence.
- **Review and evidence**: inspect traces, Agent Change Records, and task observations.
- **Reference**: command, workflow file, trace, evidence, and schema details.
- **Help**: troubleshooting and recovery.
