# AgentPlane documentation

AgentPlane turns AI-agent changes into reviewable Git evidence: task, plan, verification, Agent
Change Record, and closure.

## What are you trying to do?

### I want to try AgentPlane locally

[Run the 90-second quickstart](user/setup).

### I use Claude Code, Codex, Cursor, or Aider

[Pick a workflow guide](workflow-guides).

### I review agent-generated PRs

[Add task evidence and ACR validation](user/agent-change-record).

### I need to understand the evidence format

[Read Agent Change Record](user/agent-change-record).

### I am operating AgentPlane in a team repo

[Use branch_pr and CI gates](workflow-guides/branch-pr).

### I am upgrading an existing workspace

[Read upgrade and recovery](help/legacy-upgrade-recovery).
