# @agentplane/testkit

Internal test utilities for the AgentPlane workspace.

This package is private and is not published to npm. It supports repository tests for the CLI, core
package, recipes runtime, and workflow fixtures.

End users should install the CLI instead:

```bash
npm install -g agentplane
```

## Public packages

- `agentplane`: https://www.npmjs.com/package/agentplane
- `@agentplaneorg/core`: https://www.npmjs.com/package/@agentplaneorg/core
- `@agentplaneorg/recipes`: https://www.npmjs.com/package/@agentplaneorg/recipes
