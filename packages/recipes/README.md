Recipe manifest and overlay utilities used by the `agentplane` CLI.

End users normally install the CLI instead:

```bash
npm install -g agentplane
```

Library consumers can install this package directly when they need recipe parsing and overlay helpers:

```bash
npm install @agentplaneorg/recipes
```
